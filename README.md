# Price Drop AI Agent

## Overview
An autonomous AI agent that monitors product prices and sends email alerts when a price drop is detected.

## Features
- Autonomous execution using GitHub Actions
- Price comparison logic
- Secure email notifications
- No manual intervention required

## Tech Stack
- Python
- GitHub Actions
- Gmail App Passwords
- GitHub Secrets

## How It Works
1. Agent reads price data
2. Detects price changes
3. Makes a decision
4. Sends notification automatically

## Use Cases
- E-commerce price tracking
- Deal alerts
- Wishlist monitoring

## Future Enhancements
- Live scraping
- ML-based prediction
- WhatsApp / SMS alerts
